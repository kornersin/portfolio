Taikonauta

currently redesigning
Prototype

Objective:

Develop an educational app for children in basic education, which encourages them to study.


Problem to solve:

Current educational methods generate cognitive load and negative feelings towards studying.

Users:

Children in elementary school levels from 1st to 6th grade.


Contributions:

UX: Problem and hypothesis statement, create a proto person, insights, user journeys, empathy map, Sketch, wireframes, user test (real).

UI: Color palette selection, icon selection, prototyping, flow connection, flow review, create layouts, typography selection.



Solution:

An app based on gamming is developed where users can play and develop their skills and knowledge acquired in school, through the game they advance through different levels and subjects.
